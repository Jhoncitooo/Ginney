# Ginnery ✨

Este repositorio contiene una invitación web especial e interactiva diseñada con cariño.

## Características
- Diseño premium con efecto de cristal (Glassmorphism).
- Burbujas bokeh flotantes interactivas de fondo.
- Un botón juguetón que esquiva el cursor para evitar el "No".
- Animación de celebración con música y confeti al presionar "Sí".

## Cómo ver la página
Si estás visitando este repositorio en GitHub, puedes ver la invitación en vivo activando **GitHub Pages** en la configuración del repositorio (`Settings > Pages`) y seleccionando la rama `main` (directorio `/root`). 

Una vez activo, el enlace de GitHub Pages cargará directamente el archivo `index.html` mostrando la tarjeta de invitación interactiva.
